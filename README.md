### 设计
本项目拆分为form-design-antd与form-preview-design，前者用于表单设计编辑，后者用于表单预览（依托于现有的json格式）。

### 使用场景
初版用于基础表单设计,可将生成的json保存为不同模板，用于快速构建antd的表单
